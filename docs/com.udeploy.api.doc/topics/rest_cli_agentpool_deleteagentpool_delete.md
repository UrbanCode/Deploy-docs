# Delete an agent pool

## Request

```
DELETE https://{hostname}:{port}
    /cli/agentPool/deleteAgentPool?{parameters}

```

|Parameter|Type|Required|Description|
|---------|----|--------|-----------|
|pool|string|true|Name or ID of the agent pool|
