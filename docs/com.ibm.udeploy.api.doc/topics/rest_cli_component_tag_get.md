# Return all tags on a component

## Request

```
GET https://{hostname}:{port}
    /cli/component/tag?{parameters}
Accept: application/json

```

|Parameter|Type|Required|Description|
|---------|----|--------|-----------|
|component|string|true|Name or ID of the component|
