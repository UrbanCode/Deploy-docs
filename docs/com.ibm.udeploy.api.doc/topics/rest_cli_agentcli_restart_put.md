# Restart an agent

## Request

```
PUT https://{hostname}:{port}
    /cli/agentCLI/restart?{parameters}
Accept: application/json

```

|Parameter|Type|Required|Description|
|---------|----|--------|-----------|
|agent|string|true|Name or ID of the agent|
